## useLocalStorage(key, initialValue)

### Level: Medium, Time: 15 Minutes

### Description

- Create a custom hook that takes a key and an initial value to start with.
  - if the key does not exist in LS, we will set it to value
  - return a setter fn to update the state and also the LS
  - if key already exists in LS then don't use initial value but use the value from LS
