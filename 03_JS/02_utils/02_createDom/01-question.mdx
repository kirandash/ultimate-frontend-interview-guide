## createDom(root)

### Level: Easy, Duration: 5 Minutes, Languages: JS, TS, Frequency: 1/5

### Description

- You are given a root object which represents root node of a dom tree. The root obj will have 3 attributes:
  - type: ex: div, h1, h2 etc
  - attributes: obj that has all the attributes ex: id, class, data attributes etc
  - children: child elements or string ie text content
