## Binary Search

### Level: Medium, Duration: 20 Minutes, Languages: JS, TS

### Description

Implement a binary search fn that performs a search on an array of numbers. Assume that the input array is sorted and is an array of integers. If the target is not found in the input array then the fn should return -1, else if found then return the index of the target element.

### Examples

```
console.log(binarySearch([3, 6, 8, 9, 56], 56)); // 4
console.log(binarySearch([3, 6, 8, 9, 56], 10)); // -1
```
